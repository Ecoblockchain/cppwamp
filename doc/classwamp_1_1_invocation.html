Detailed Description

Provides the means for returning a YIELD or ERROR result back to the RPC caller.

Examples:
examples/chat/main.cpp.

bool wamp::Invocation::isProgressive ( ) const

Checks if the caller requested progressive results.

This function checks if the INVOCATION.Details.receive_progress|bool detail is true. See Progressive Call Results in the advanced WAMP spec.

Variant wamp::Invocation::caller ( ) const

Returns an optional session ID integer of the caller.

This function checks the value of the INVOCATION.Details.caller|integer detail. See Caller Identification in the advanced WAMP spec.

Returns
An integer variant if the caller ID is available. Otherwise, a null variant is returned.
Variant wamp::Invocation::trustLevel ( ) const

Returns an optional trust level integer.

This function checks the value of the INVOCATION.Details.trustlevel|integer detail. See Call Trust Levels in the advanced WAMP spec.

Returns
An integer variant if the trust level is available. Otherwise, a null variant is returned.
Variant wamp::Invocation::procedure ( ) const

Returns an optional string of the original procedure URI used to make this call.

This function checks the value of the INVOCATION.Details.procedure|uri detail. See Pattern-based Registrations in the advanced WAMP spec.

Returns
A string variant if the procedure URI is available. Otherwise, a null variant is returned.
