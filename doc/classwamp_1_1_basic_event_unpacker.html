Detailed Description

template<typename TSlot, typename... TArgs>
class wamp::BasicEventUnpacker< TSlot, TArgs >

Wrapper around an event slot which automatically unpacks positional payload arguments.

The wamp::basicEvent convenience function should be used to construct instances of BasicEventUnpacker. This class differs from EventUnpacker in that the slot type is not expected to take an Event as the first parameter.
