class wamp::Codec

template<CodecId id>
class wamp::Codec< id >

Empty default implementation of the Codec template.

Codecs are static-only classes that can perform serialization from, and deserialization to Variant objects. Codec is a template class that is specialized for each CodecId enumerator. Specializations must conform to the Serializer concept.
