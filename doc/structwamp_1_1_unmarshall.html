Detailed Description

template<typename... TArgs>
struct wamp::Unmarshall< TArgs >

Utility class used to split Array elements into separate arguments to be passed to a function.

This class is used internally by Session to invoke event slots and call slots taking extra, statically-typed parameters.

Template Parameters
TArgsList of target argument types to which the Array elements will be split and converted.
