Detailed Description

template<>
struct wamp::Codec< CodecId::msgpack >

Codec specialization for Msgpack serialization.

This static-only class is used to serialize/deserialize between Msgpack payloads and Variant objects.

Type Requirements
Meets the requirements of the Serializer concept.
Note
The more readable Msgpack alias should be used instead of Codec<CodecId::msgpack>, wherever possible.
