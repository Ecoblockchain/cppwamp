Detailed Description

template<typename TDerived, typename TProtocol>
class wamp::RawsockOptions< TDerived, TProtocol >

Base class containing general options for raw socket connections.

Note
Support for these options depends on the socket protocol, as well as the operating system. For example withDoNotRoute probably has no effect on Unix domain sockets.
Template Parameters
TDerivedType of the class that derives from this one. Used for method chaining.
TProtocolThe Boost Asio protocol type these options apply to.
