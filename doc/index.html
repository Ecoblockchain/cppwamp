CppWAMP Documentation

This is the reference documentation for the CppWAMP library. CppWAMP is a C++11 implementation of the WAMP protocol's client roles (caller, callee, publisher, and subscriber).

The CppWAMP open-source project is hosted at https://github.com/ecorm/cppwamp, and is released under the Boost License.

For an overview of the library's classes, structs, typedefs, enums, etc., consult the wamp Namespace Reference.

For a tutorial-style overview, please consult the Tutorial wiki page on GitHub.

For an example chat program, please see examples/chat/main.cpp.

For a brief architectural overview of the library, see Architectural Overview.
